This plugin will delete post of any post type with easy way.

== Description ==

If you want to delete your all post at a time this plugin will help you to do this.
Go to Delete Post settings page selece a post type then click on start delete button.
Done!

== Installation ==

1. Upload `delete-post-swrs` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Delete Post settings page under the Settings menu of your Dashboard
